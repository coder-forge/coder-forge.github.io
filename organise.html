Organising a coder forge is a 2 step process

  1. Find a venue
  2. Advertise the forge
The coder forge project is released under the creative commons 4.0 license, this means that more forges can grow holistically by simply forking the repository. The license in use also allows for different forges to take different directions. Github Repository
